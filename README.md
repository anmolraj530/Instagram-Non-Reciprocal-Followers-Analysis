# Instagram Non-Reciprocal Followers Analysis (MySQL)

This project identifies Instagram accounts that a user follows but which do not follow back using MySQL.

## Tools Used
- MySQL
- SQL LEFT JOIN
- Synthetic CSV datasets

## Steps to Run
1. Run SQL scripts in order from the sql/ folder
2. Import CSV files from the data/ folder
3. Execute the analysis query

## Output
List of users that do not follow back.
